## 🔧 技术架构

### 模块结构
```
psoc/
├── src/
│   ├── i18n/                   # 国际化模块 🆕
│   ├── ui/                     # 用户界面
│   ├── tools/                  # 工具系统
│   ├── commands/               # 命令系统
│   ├── rendering/              # 渲染引擎
│   └── ...
├── resources/
│   └── i18n/                   # 语言资源文件 🆕
│       ├── en.ftl              # 英文翻译
│       └── zh-cn.ftl           # 中文翻译
└── tests/
    ├── i18n_tests.rs           # i18n测试 🆕
    └── ...
```

### 核心技术栈
- **语言**: Rust (稳定版)
- **GUI框架**: iced 0.13.1
- **国际化**: Fluent 0.16.1 🆕
- **图像处理**: image crate
- **序列化**: serde + ron
- **测试**: 内置测试框架
